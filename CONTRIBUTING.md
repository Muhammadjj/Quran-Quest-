# Contributing to Quran Quest


Thank you for showing interest in contributing to **Quran Quest**.  
This project is open-source so that the community can help improve it.

---

## How to Contribute

1. **Fork the Repository**  
   - Click the "Fork" button at the top right of this repo.  
   - Clone your fork locally:
     ```bash
     git clone https://github.com/<your-username>/quran_quest.git
     ```

2. **Create a New Branch**  
   Always work on a new branch for your changes:
   ```bash
   git checkout -b feature/my-new-feature
